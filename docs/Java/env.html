mavenとは¶

POM(Project Object Model)という考え方に基づいて、プロジェクトのビルド、テスト、ドキュメンテーション、成果物の配備などプロジェクトのライフサイクルを管理するもの。 プロジェクトに関わる情報はPOMに集約する。

  • 参考　Mavenとは何ぞや
